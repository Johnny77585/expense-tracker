# 記帳軟體

![Index page about Restaurant List](./public/image/snapshot.jpg)

## 介紹

記錄自己的每日支出，提高對於金錢的掌握度!

### 功能

- 首頁查看所有紀錄
- 新增紀錄
- 刪除紀錄
- 編輯紀錄
- 篩選類別
- 篩選日期
- 查詢紀錄名稱
- 按照金額or日期排序

## 開始使用

1. 請先確認有安裝 node.js 與 npm
2. 將專案 clone 到本地
3. 在本地開啟之後，透過終端機進入資料夾，輸入：

   ```bash
   npm install
   ```

4. 安裝完畢後，繼續輸入：

   ```bash
   npm run start
   ```

5. 若看見此行訊息則代表順利運行，打開瀏覽器進入到以下網址

   ```bash
   Listening on http://localhost:3000
   ```

6. 若欲暫停使用

   ```bash
   ctrl + c
   ```
